## 2. Missing Audacity/Ableton QoL — recommended next

De-duplicated from the two tool lists (overlapping items merged: time-range selection, follow-playhead, arrow nudge, context menus, markers, loop region, volume automation). Grouped quick-wins-first.

### Tier 1 — Quick wins (small effort, high leverage)

| Feature                                           | Tool(s)  | Benefit                                                                                                                 | Effort    |
| ------------------------------------------------- | -------- | ----------------------------------------------------------------------------------------------------------------------- | --------- |
| **Follow playhead / scroll-follow**               | Both     | View auto-scrolls to keep playhead visible on zoomed sessions; position polling already exists, just drive `scrollSec`. | small     |
| **Right-click context menus** (clip/track/ruler)  | Both     | Split/duplicate/delete/rename/color where the pointer is; no `onContextMenu` exists today.                              | small–med |
| **Arrow-key nudge** of selected clip              | Both     | Precise, repeatable placement by grid step; keydown handler currently ignores arrows.                                   | small     |
| **Snap enable/disable toggle + momentary bypass** | Both     | Free-hand placement near clip edges without fighting the always-on snap.                                                | small     |
| **Fit-to-window** (zoom to fit project)           | Audacity | One-click overview after zooming; only incremental zoom exists now.                                                     | small     |
| **Zoom-to-selection**                             | Audacity | Frame the selected clip/range for detailed trims.                                                                       | small     |
| **Timeline quick-play** (click ruler to audition) | Audacity | Removes the separate Play press on every "what does this sound like".                                                   | small     |
| **Fold / unfold all tracks**                      | Ableton  | Global collapse toggle; per-track chevron exists but no session-wide one.                                               | small     |
| **Keyboard shortcuts cheat-sheet** (`?` overlay)  | Both     | Makes the existing buried shortcuts discoverable/trustworthy.                                                           | small     |
| **Zoom-adaptive beat-grid**                       | Both     | Grid density follows zoom; time-grid already adaptive, beat-grid still a manual dropdown.                               | small     |

### Tier 2 — Medium effort, high value

| Feature                                                | Tool(s)  | Benefit                                                                                                                 | Effort    |
| ------------------------------------------------------ | -------- | ----------------------------------------------------------------------------------------------------------------------- | --------- |
| **Loop region / loop brace UI**                        | Both     | Audition-on-repeat while editing. **Backend `play()` already accepts loopStart/loopEnd** — mostly UI + wiring.          | med       |
| **Snap-to-zero-crossing**                              | Audacity | Prevents clicks/pops at cut boundaries — **directly serves the release-blocking FR-2.3 clean-capture goal.**            | med       |
| **Time-range selection** (marquee across ruler/tracks) | Both     | The keystone primitive: enables silence/delete/trim/consolidate/loop/export-selection. Waver is single-clip-only today. | med–large |
| **Normalize / Amplify**                                | Audacity | One-click peak-to-target; implementable non-destructively via existing peak pyramid → clip gain.                        | med       |
| **MP3 / Ogg / Opus export**                            | Audacity | Compressed share format; today only lossless WAV/FLAC.                                                                  | med       |
| **Marker / locator track**                             | Both     | Named positions (Verse/Chorus), jump + snap anchors; no markers exist.                                                  | med       |
| **Audible metronome / count-in**                       | Both     | Grid+BPM already exist but are silent; lets performers track to tempo.                                                  | med       |
| **Undo history panel**                                 | Audacity | Multi-step jump-back vs. blind repeated ⌘Z; can_undo/redo already tracked.                                              | med       |
| **Duplicate Time / Insert / Delete Time**              | Ableton  | Ripple a vertical slice across all tracks for arrangement building.                                                     | med       |

### Tier 3 — Large effort, deferred

| Feature                                             | Tool(s)  | Benefit                                                                                | Effort |
| --------------------------------------------------- | -------- | -------------------------------------------------------------------------------------- | ------ |
| **Volume envelope / gain automation** (breakpoints) | Both     | Time-varying level rides/ducks beyond static per-clip gain.                            | large  |
| **Crossfades on overlapping clips**                 | Both     | Seamless comps/splices; engine currently prevents overlap (`freeStartOn` bumps clips). | large  |
| **Consolidate** (render selection to one clip)      | Both     | Inverse of split; tidies fragmented tracks.                                            | large  |
| **Spectrogram view**                                | Audacity | Reveals hum/noise/clipping for clean-capture verification.                             | large  |
| **Group tracks** (bus/fold)                         | Ableton  | Manage stems/multi-mic as one unit.                                                    | large  |
| **Warp markers** (time-stretch)                     | Ableton  | Align takes to grid; needs real-time DSP — stretch goal.                               | large  |

### Top 5 to do next

1. **Follow playhead / scroll-follow** — small, plumbing already there, fixes losing your place on any zoomed take.
2. **Loop region UI** — high value, backend loop args already wired; mostly draw-the-brace + pass the range.
3. **Snap-to-zero-crossing** — medium, but the only listed gap that directly advances the release-blocking FR-2.3 clean-capture requirement.
4. **Right-click context menus** — small, biggest single discoverability/speed jump for existing commands.
5. **Arrow-key nudge + snap toggle** — small, together give precise, controllable placement that dragging can't.
